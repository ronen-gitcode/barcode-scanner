# barcode-scanner

דף סריקת ברקודים (1D) למצלמת הטלפון, חלק ממערכת ספירת מלאי.
**מאגר זה ציבורי ומכיל קוד גנרי בלבד — אין בו נתונים ואין בו סודות.**

הדף סורק ברקוד וכותב אותו כקובץ קטן למאגר ה-GitHub הפרטי `barcode` באמצעות מפתח גישה
שהמשתמש מדביק בטלפון (נשמר רק בדפדפן המכשיר, לא בקוד).

## פרסום
`Settings` → `Pages` → Deploy from a branch → `main` / `root`.
הכתובת המתקבלת: `https://<user>.github.io/barcode-scanner/`
